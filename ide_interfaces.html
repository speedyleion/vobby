IDE Interfaces¶

Vobby utilizes a generic interface for communicating between different IDE interfaces. This is done by implementing the FileBuffer class. This extra abstraction allows for possible expansion to other protocols outside of Infinoted as well as possibly using the Vim/Netbeans implementation elsewhere.

class file_buffer.FileBuffer[source]¶

Bases: object

This is an abstract class representing a file buffer for an editor. The idea is that this class provides a generic interface for describing buffer modifications without having to know the nitty gritty details of a specific editor interface.

This will delete the length number of characters from the offset in this buffer.

Parameters:
  • offset (int) – This is the number of characters into the buffer to place content at. This is a 0 based value. Newlines are considered one character. TODO: do we need to worry about ‘rn’?

Kwargs:

The main purpose of this is for editor interfaces that support highlighting text by different users as different colors.

sync(content)[source]¶

This is usually used on initial creation/editing of a shared buffer. content will be the entire contents of the buffer. Most implementations will probably remove all of the previous contents in the buffer and replace them with this new content.
